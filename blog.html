Important FAQ

What is the Document Objext Model (DOM)?

The DOM is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each node is an element (e.g., HTML tags, text, attributes). Developers use the DOM to interact with, modify, and update the content, structure, and styles of a webpage dynamically.

How do you select an element from the DOM?

You can select an element from the DOM using various JavaScript methods, such as:

  • document.getElementById('id') – Selects an element by its ID.
  • document.querySelector('.class') – Selects the first element that matches a CSS selector.
  • document.querySelectorAll('selector') – Selects all elements matching a CSS selector.
  • document.getElementsByClassName('className') – Selects elements by class name.
  • document.getElementsByTagName('tag') – Selects elements by tag name.

Waht is even delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element instead of adding listeners to each individual child element. This leverages event bubbling (the process where events propagate up through the DOM hierarchy) to handle events for multiple elements. It's useful because:

  • It reduces the number of event listeners, improving performance.
  • It allows you to manage dynamically added elements without needing to reassign event listeners.
How do you manipulate an elemet's attributes and styles using the DOM?

To manipulate attributes, you can use:

  • element.setAttribute('attribute', 'value') – Sets an attribute value.
  • element.getAttribute('attribute') – Retrieves an attribute value.
  • element.removeAttribute('attribute') – Removes an attribute.

To manipulate styles, you can modify the style property:

  • element.style.property = 'value' – For example, element.style.color = 'blue';.
